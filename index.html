About Data Analysis Report

This RMarkdown file contains the report of the data analysis done for the project on forecasting daily bike rental demand using time series models in R. It contains analysis such as data exploration, summary statistics and building the time series models. The final report was completed on Thu Jul 6 18:19:33 2023.

Data Description:

This dataset contains the daily count of rental bike transactions between years 2011 and 2012 in Capital bikeshare system with the corresponding weather and seasonal information.

Data Source: https://archive.ics.uci.edu/ml/datasets/bike+sharing+dataset

Relevant Paper:

Fanaee-T, Hadi, and Gama, Joao, ‘Event labeling combining ensemble detectors and background knowledge’, Progress in Artificial Intelligence (2013): pp. 1-15, Springer Berlin Heidelberg

Task Three: Smooth time series data

Converting columns to vectors

dates <- day$dteday
count <- day$cnt

Locating Outliers

#creates time series object
bikets <- ts(day$cnt, start = dates, frequency = 7)
plot(bikets)

#identifies and replaces outliers
cleanbikes <- tsclean(bikets, replace.missing = TRUE)
plot(cleanbikes)

####Smooth time series data using Holt-winter’s additive method

fit <- hw(cleanbikes, seasonal = "additive")
plot.ts(cleanbikes, main = "Smoothed Timeseries of Bike Rentals", col = "blue")
lines(fitted(fit), col = "red")

####Smooth time series data using Simple Moving area

cleanMA10 <- SMA(cleanbikes, n = 10)
plot.ts(cleanMA10)

Task Six: Findings and Conclusions

Through this process it can be seen that there is a positive correlation between the months in the years and the number of bikes that are rented.

The data also supports that there is a positive trend between temperature and bike rentals where the median number of bikes rented were higher when temps were in the range of 68 - 75 degrees.

An ARIMA and linear regression model were used to get an estimate of bikes to be rented in the next 25 intervals (months). Some takeaways to note, outliers and noise in the data were smoothed to estimate the next bike rental amounts.

Further research that could be conducted is to examine trends in holiday rentals and potential analysis on hourly use of bikes.Data also pointed towards most of the bike rentals being made between late Spring, Summer, and early Fall seasons.
